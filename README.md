# Weather Dashboard 🌦

A modern weather dashboard that displays real-time weather information for any city using the OpenWeatherMap API.

## Features

- Search weather by city name
- Displays temperature
- Shows humidity and wind speed
- Air Quality Index (AQI) with color indicators
- Dynamic background based on weather conditions
- Responsive modern UI design

## Technologies Used

- HTML
- CSS
- JavaScript
- OpenWeatherMap API

## API Used

OpenWeatherMap API is used to fetch:
- Weather data
- Air pollution (AQI)

## How to Run

1. Clone the repository
2. Open the project folder
3. Add your OpenWeatherMap API key in `script.js`
4. Open `index.html` in a browser
